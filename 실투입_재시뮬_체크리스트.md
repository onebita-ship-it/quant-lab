# 실투입 재시뮬 체크리스트

> **목적**: 실제 내 돈을 투입하기로 결정한 시점에, 지금 확정된 B안이 **여전히 최선인지 그때 데이터로 다시 검증**하고,
> 후보들과 재비교해 **그 시점의 최고 전략**으로 개시하기 위한 절차서.
> **언제 쓰나**: "이제 진짜 투자한다" 결정이 섰을 때. 그 전까지는 열어볼 필요 없음.
> **작성**: 2026-07-14 (Codex 교차검증 세션 후). 현재 확정안은 2026-07-04 데이터 기준 챔피언일 뿐임.

---

## 왜 다시 돌리나 (핵심 전제)

- 지금 `result_final.md §0`의 **B안(코어 42.5 / 위성 42.5 / 금 15)**은 **2026-07-04까지 데이터** 기준 최적안이다.
- 실투입이 몇 달~몇 년 뒤라면 시장·모멘텀·국면이 달라져 **그때 챔피언이 바뀔 수 있다.**
- 그러니 "예전 결론"을 그대로 믿지 말고 **최신 데이터로 전 과정을 재실행 → 재비교 → 재확정**한다.
- ⚠️ **투트랙 방화벽**: `§0`(운용 룰) 변경은 *분기 리뷰 + 재검증 통과* 시에만. 이 체크리스트가 곧 그 재검증 절차다.

## STEP 1 — 최신 데이터로 갱신

```bash

.venv/bin/python scripts/download_data.py --force            # 실시세 최신화 (인터넷 필요)
.venv/bin/python scripts/download_data.py --force --candidates # 후보 3배 ETF + 원지수까지
# ⚠️ 합성 5종 전부 재생성해야 함(하나라도 빠지면 STEP2/3에서 *_SYNTH FileNotFound). 2026-07-17 실경험.
.venv/bin/python scripts/make_synthetic_tqqq.py              # QQQ→가상 TQQQ (닷컴·GFC 구간)
.venv/bin/python scripts/make_synthetic_benchmark.py         # TLT/TMF/GLD/IEF/AGG/EFA 합성(벤치·금오버레이)
.venv/bin/python scripts/make_synthetic_candidates.py        # UPRO/SOXL/SQQQ/SGOV 합성(위성 v8/v9)
.venv/bin/python scripts/make_synthetic_candidates2.py       # SHY/TIP/VEU/EEM/SCZ/LQD 합성(후보전략)
.venv/bin/python scripts/make_synthetic_universe2.py         # TECL/TNA/EDC/YINN/KORU 합성(유니버스 감사)
```
- [ ] `data/` 캐시가 오늘 날짜로 갱신됐는지 확인. `ls data/*_SYNTH*`로 합성 23종 생성 확인.
- [ ] 유니버스 편입/제외 검토 (`config/universe.txt` — 분기 리뷰 재량).

## STEP 2 — 확정안(B안) 재검증

> 각 스크립트의 정확한 플래그는 실행 시 `--help` 또는 파일 상단 주석으로 확인(시간이 지나면 옵션이 바뀌었을 수 있음).

```bash

.venv/bin/python backtests/final_audit_crossval.py   # 교차검증 + 카나리아(13612W) 실판정
.venv/bin/python backtests/portfolio_backtest.py     # 코어/위성/금 3슬리브 배분 백테스트
.venv/bin/python simulations/monte_carlo_final.py --years 5 --paths 2000     # 미래 분포
.venv/bin/python simulations/monte_carlo_portfolio.py --years 5 --paths 2000 # 포트폴리오 MC
```
- [ ] 견고성 확인: 파라미터(분할40·익절15%·SL20·스트릭5)가 여전히 **완만한 고원**인가? (뾰족하면 과최적화 경고)
- [ ] MDD/반토막 확률이 감내 가능 범위인가? (현재 기준: MDD -54% 각오, 반토막 2.9%)
