# 무한매수법 v4 — 쿨다운 "하락 국면 한정" 검증 리포트

> 작성일: 2026-07-04
> 데이터: `TQQQ_SYNTH` (QQQ 기반 합성 TQQQ, 1999-03-10 ~ 2026-07-02)
> 결론 미리: **v4는 v3의 CAGR 붕괴를 정확히 제거했고, 그 과정에서 "쿨다운은 필터와 중복이라
> 사실상 불필요"하다는 것을 증명했다.** 최종 견고안은 v3=v4의 "기울기+스트릭 필터, 쿨다운 OFF".

## 1. 문제 (v3에서 넘어온 과제)

v3의 손실 후 재진입 쿨다운(C)을 **상시** 적용하니, 상승장의 정상적 되돌림에서 난 소진 손실에도
20일씩 시장을 비워 복리를 놓쳤다 → **전체구간 CAGR 14.0% → 4.6%로 붕괴.**
과제: **"추세가 하락 중일 때만 쿨다운을 켜서"** 상승장 참여는 지키고 폭락장 방어만 얻자.

## 2. v4 구현 — `strategies/infinite_buying_v4.py`

- **하락 국면 플래그(`regime_down`)**: **50일선 < 200일선(데드크로스 레짐)**. 진입 필터
  (가격>200MA·200MA 상승)보다 **느린** 신호라, 급락장 중 브리프 랠리로 진입 필터가 잠깐 켜져도
  레짐은 계속 '하락'으로 유지되도록 설계.
- **조건부 쿨다운**: 손실 사이클 종료 시 `cooldown_when_down_only=True`면 **그 시점이 하락 국면일
  때만** N일 진입 금지. 상승 국면 손실은 쿨다운 없음.
- (A)기울기 필터·(B)확정 스트릭5·쿼터손절·현금비중은 v3 그대로 계승.

## 4. 핵심 발견 — v4 = v3 무쿨다운 (쿨다운은 no-op)

**v4(하락시만 쿨다운)는 v3 무쿨다운과 전 구간·전 지표에서 완전히 동일하다.** 즉 하락 국면
한정 쿨다운은 단 하나의 진입 결정도 바꾸지 못했다. 계측으로 이유를 확인:

| 계측 항목 | 값 |
|-----------|----|
| 진입 필터 허용 비율 | 66.6% |
| 하락 국면(50<200) 비율 | 25.6% |
| **손실 소진 사이클 — 상승 국면** | **35건** |
| **손실 소진 사이클 — 하락 국면** | **7건** |

**쿨다운의 두 얼굴**:
- **상승 국면 손실(35건)**: 상시 쿨다운은 여기서 발동해 회복하는 상승장 재진입을 막음 →
  이것이 v3의 CAGR 붕괴(14%→4.6%)의 정체. v4는 하락 한정이므로 이 35건에 **발동하지 않음** → CAGR 회복.
- **하락 국면 손실(7건)**: v4 쿨다운이 발동하지만, 하락 국면에선 **진입 필터(가격>200MA·상승)가
  이미 재진입을 차단**하고 있어 쿨다운이 막을 것이 없음 → **효과 0(중복)**.

➡️ 결론: 쿨다운은 상승장에선 해롭고, 하락장에선 필터와 중복이다. **(A)기울기+(B)스트릭 진입 필터가
이미 하락 국면 방어를 충분히 수행하므로, 쿨다운은 어떤 형태로도 순이득이 없다.** v4의 성과는
"쿨다운을 하락 한정으로 만들어 무해화"한 것이며, 실질 최적은 **쿨다운 OFF(=v3 무쿨다운)**.

## 5. 최종 권고 (v1~v4 종합)

- **최종 견고안: v3/v4 "기울기 필터 + 확정스트릭5" (분할40/익절15%/쿼터손절), 쿨다운 OFF, 100%.**
  → 전체 CAGR 14.0%(≈v1)·샤프 0.59·MDD -62%, 닷컴 -38.8%, GFC -38.2%.
  v1의 -95% 파멸 리스크를 거의 수익 손실 없이 실전 감내 수준으로 낮춤.
- **추가 방어가 필요하면 현금비중을 낮춰라(쿨다운이 아니라).** 현금 75% 투입 시 전체 MDD -52.2%·
  샤프 0.60(최고), 닷컴 -29.1%, GFC -28.6%. 국면 무관하게 작동하는 유일하게 깨끗한 리스크 레버.
- **쿨다운(C)은 폐기 권고**: 상승장에선 CAGR 훼손, 하락장에선 필터와 중복. 진입 필터가 이미
  타이밍 방어를 담당하므로 별도 쿨다운 계층은 불필요.

> ⚠️ 스트레스 구간의 TQQQ_SYNTH는 QQQ 합성값이며 실제 3배 ETF의 일일 리밸런싱과 완전히 같지 않음.
> 방향성 참고용. 실데이터·다른 자산 교차검증 권장.

## 6. 몬테카를로 — 최종 견고안 (5년 × 2000경로)

최종 견고안(**기울기+스트릭5 · 쿼터손절 · 쿨다운 OFF**)을 미래 시뮬레이션. 파이프라인은
QQQ 20일 블록 부트스트랩 → 각 경로 QQQ로 추세신호 → 같은 QQQ에서 3x TQQQ 파생 → 전략 실행
(200일선 워밍업으로 실제 QQQ 최근 250일 부착). 권장 추가방어 옵션(현금 75%)을 동일 경로에서 병기.

| 지표 | 최종 (100% 투입) | 최종 + 현금 75% |
|------|---------------:|---------------:|
| 총수익 5%ile | -56.6% | **-42.5%** |
| 총수익 중앙값 | **31.5%** | 23.7% |
| 총수익 95%ile | **342.4%** | 256.8% |
| MDD 중앙값 | -48.9% | **-40.1%** |
| MDD 5%ile(악화) | -73.0% | **-59.7%** |
| 손실 확률(원금 미만) | 35.4% | 35.4% |
| 반토막 확률(-50%↓) | 7.8% | **2.0%** |

**해석**
- **최종 견고안(100%)**: 중앙값 5년 +31.5%. 그러나 하위 5% -56.6%·최악 MDD -73%로 3배 레버리지
  꼬리위험은 여전. 중앙 경로에서도 MDD -49%(반토막 근처)를 통과. (이 100% 수치는 `result_v3.md`
  §6의 '추세 ON' 결과와 일치 — 동일 구성이므로 일관성 확인됨.)
- **현금 75% 추가방어**: 상단(중앙값 31.5→23.7%)을 반납하는 대신 **하방 꼬리를 크게 개선** —
  반토막 확률 **7.8% → 2.0%**, 최악 MDD **-73% → -60%**, 5%ile 총수익 -56.6% → -42.5%.
- **손실 확률은 두 구성 모두 35.4%로 동일**: 유휴 현금은 손실 경로의 *부호*를 못 바꾸고 *크기*만
  줄이기 때문(75% 투입이면 총수익 = 0.75 × 전략수익). 즉 현금비중은 "지는 걸 막는" 게 아니라
  "질 때 덜 지게" 하는 레버 → §5 결론(추가 방어는 쿨다운 아닌 현금비중)과 정합.
- ⚠️ **한계**: 20일 블록 부트스트랩은 닷컴·GFC 같은 다년 지속 하락장을 거의 못 만들어 추세 필터의
  이점을 **과소평가**한다. 필터의 진짜 값은 랜덤 미래가 아니라 §3의 실제 역사 스트레스에서 드러남.

> ⚠️ "과거 수익률 분포 유지" 가정. 2000·2008급 폭락이 더 잦거나 크면 위 수치보다 위험이 크다.
