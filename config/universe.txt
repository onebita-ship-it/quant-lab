# v9 유니버스 — 사람이 분기마다 손으로 관리하는 파일(재량).
#
# 형식:  티커  구분  원지수
#   - 티커   : 매매 대상(레버리지 ETF). 데이터는 {티커}_SYNTH.csv 우선, 없으면 {티커}.csv.
#   - 구분   : core | satellite
#              · core      = 엔진B(폭락 계단투입) 대상. 원지수 대폭락 시 여기서만 물타기.
#              · satellite = 엔진A(추세+모멘텀 로테이션)만 참여. 폭락 물타기 대상 아님.
#   - 원지수 : 추세신호(200일선 기울기+스트릭5) 계산에 쓸 기초지수 티커.
#
# 규칙: 유니버스에 새 티커를 추가해도 '그 티커의 원지수 추세신호가 ON'이 되기 전엔 매수되지 않는다
#       (엔진A가 신호 통과 자산만 후보로 삼기 때문). 분기 리뷰 때 이 파일만 고치면 전략이 따라온다.
#
# '#' 로 시작하는 줄과 빈 줄은 무시.
